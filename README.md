# EDS 241 Final Project

## About

This repository contains the final project materials for the course EDS 241 - Environmental Policy Evaluation at UCSB's Masters of Environmental Data Science. The project's main goal is to recreate the statistical methods done by **Mildenberger et al., 2023: Unintended consequences of using maps to communicate sea-level rise**.

## Data

- The authors provided the data they used directly from their paper in the *Data Availability* section and is housed in a [Harvard Dataverse Repository](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/1QCDCG).

## References

- Mildenberger, M., Sahn, A., Miljanich, C. et al. Unintended consequences of using maps to communicate sea-level rise. Nat Sustain 7, 1018–1026 (2024). https://doi.org/10.1038/s41893-024-01380-0

- Sahn, Alexander. “Replication Data for: Using Maps to Communicate Sea-Level Rise Can Undermine Concern among Risk-Exposed Populations.” With Alexander Sahn, Harvard Dataverse, 2024. DOI.org (Datacite), https://doi.org/10.7910/DVN/1QCDCG.
